## Suma 
Para sumar es tan simple como
1 + 1 = 0 , CARRY = 1 
1 + 0 = 1 , CARRY = 0
0 + 1 = 1 , CARRY = 0
0 + 0 = 0 , CARRY = 0

## Resta
Para restar es lo mismo que la suma pero al numero negativo lo complementamos a la base y luego sumamos el numero complementado. [[Complemento a la base]]
